Класс UpdateValue изменяет словари, списки.   Заменяет одно или несколько значений   в списке.

update_inner_value. Этот Метод меняет значеня заданных ключей в словаре.Создает новые ключи.Меняет значения самих ключей.       Словарь изменяется.

Принимает на вход словарь, список и значение
Если в списке один элемент, меняется значение ключа на первом уровне вложенности
Если такого ключа нет, он создается
Если в списке несколько элементов:
последний меняет значение этого ключа
Если такого ключа нет, он создается
Остальные элементы - это ключи
Мы проваливаемся до нужного уровня вложенности

replacing_items_in_a_list. Этот Метод заменяет одно значение в списке на новое.Список не меняется.

replacement_by_specific_indices_list. Этот Метод заменяет несколько элементов в списке на одно заданное значение по заданным индексам.Список не меняется.

multiple_different_substitutions_list. Этот Метод заменяет несколько элементов в списке на определенные значения по заданным индексам.Список изменяется.

multiple_substitution_list. Этот Метод заменяет несколько элементов в списке.Список не меняется.

reverse_the_list. Переворачиваем список в обратном порядке.Список не меняется.

getting_data_by_indexes_from_a_list. Возвращает список элементов из определенного списка по заданным индексам

getting_data_by_indexes_gen. Возвращает генератор из определенного списка по заданным индексам

reverse_the_string. Переворачиваем строку в обратном порядке.

remove_dups. Удаление дубликатов из списка, сохраняя порядок.

remove_dups_dict1. удаление дубликатов в списке словарей.

remove_dups_dict2. Удаление дубликатов в списке словарей.

creating_a_set. Создание и использование Set - множества из списка.

creating_a_dict. Создание и использование Dict - словаря из списка.

get_unique_data. Возвращает список с уникальными значениями.
           Вложенный генератор с условием.

creation_of_matrix. Создание матрицы.

matrix_assembly. Сборка матрицы.

get_sum_gen. Используйте генераторы для больших наборов данных.
